## Iniciar projeto em NODE

```sh
npm init  # Inicializa um novo projeto Node.js interativamente

npm init -y  # Inicializa um novo projeto Node.js com configurações padrão

tsc --init  # Inicializa um arquivo de configuração TypeScript

npm install express  # Instala o framework Express.js

npm install cors  # Instala o pacote CORS para controle de acessos

npm install --save-dev typescript @types/node  # Instala o TypeScript e os tipos para Node.js como dependências de desenvolvimento

npm install --save-dev @types/express  # Instala os tipos para Express.js como dependências de desenvolvimento

npm install --save-dev @types/cors  # Instala os tipos para CORS como dependências de desenvolvimento

npx ts-node  # Executa arquivos TypeScript diretamente sem precisar de compilação
```
